Parabéns! Você é um verdadeiro Git Hubber! Por favor, evite se exibir para o coleguinha do lado, e não tente inspirar o mundo com sua grandeza.

Brincadeiras a parte, agradecemos a presença e esperamos que você tenha aproveitado o curso. Caso tenha interesse, ficam aqui duas recomendações de material extra sobre o assunto:

https://www.atlassian.com/git/tutorials/advanced-overview/ Tutoriais da Atlassian sobre a ferramenta. Eles também possuem tutoriais mais básicos, todos em inglês.

https://gist.github.com/jaseemabid/1321592 Coleção de vários tutoriais de Git para diversos níveis de conhecimento, além de cheatsheets e alguns vídeos interessantes. Também todo em inglês.

https://rogerdudler.github.io/git-guide/index.pt_BR.html Guia prático dos comandos mais utilizados. Disponível em PT-BR.

Algumas recomendações de GUI:

TortoiseGit (Windows)
SourceTree (Windows/Linux/Mac)
GitHub Desktop (Windows/Linux/Mac)

A maioria das IDEs de hoje possuem algum tipo de integração com Git também, então muitas vezes não será preciso utilizar uma ferramenta própria para o Git. Apesar disso, ter conhecimento das instruções em git no terminal é muito importante para o uso prático da ferramenta, visto que alguns conflitos precisam ser resolvidos manualmente.
