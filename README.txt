Você quer testar seus conhecimentos de Git no terminal? Esse jogo é feito justamente para isso! 
Você deve navegar pelos cinco níveis presentes dentro desse repositório, utilizando os comandos básicos da ferramenta para seguir as instruções presentes no README.

Cada nível possui uma charada simples que completará as instruções, te dando um nome de branch ou o comentário do commit para o qual você deve ir. Como sempre, Google é seu amigo nas horas em que você não souber direito o que fazer.

Caso precise de ajuda com os comandos você pode acessar a documentação oficial (https://git-scm.com/doc) ou utilizar o git help [command]. O jogo foi pensado para usar apenas os comandos mais utilizados do Git, então não é necessário ir muito fundo na documentação.

O projeto foi inspirado nesse repositório: https://github.com/hgarc014/git-game.git 
Fique a vontade para olhar a versão original, mas tenha em mente que ela utiliza conhecimentos mais avançados de Git.

Enfim, ao jogo! Para chegar ao nível 1, faça o checkout do commit que tem como comentário a resposta para a pergunta a seguir:

Quais as primeiras palavras de todo programador?
